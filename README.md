# YaFo-Compatible-Purchase-Assistant
YaFo-compatible purchase assistant for XML order generation, inventory cost filling, new product pricing, and sales forecasting.
# YaFo兼容采购助理

用于兼容采购业务中的工厂直发订单、分仓订单、供应商直发表格处理、新品定价、存货成本回填、销量预测与备货建议等场景。

## 当前版本

v3.5.37

## 主要功能

- 工厂直发订单 XML 生成
- 分仓订单 XML 生成
- 供应商直发表格处理
- 新品上新定价
- 存货成本回填
- 销量预测与备货建议
- NC 存货 CSV 导入

## 运行方式

```bash
python YaFo兼容采购助理_v3.5.37.py
